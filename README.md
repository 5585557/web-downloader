# 网页资源下载器

一键下载完整网页及其所有资源（CSS/JS/图片/字体等），支持离线浏览。

## 功能

- 智能代理下载 — 打开网页后自动抓取所有资源
- 同源劫持模式 — HTTP网站Cookie/Session完整保留，小游戏完美运行(注意,会检测浏览器兼容性的小游戏网站无法下载)
- 兼容代理模式 — HTTPS网站通过本地代理中转
- 缓存持久化 — 停止后重启不重复下载
- 网页翻译 — Google/MyMemory/Argos/Lingva + 多种AI翻译
- 批量下载 — 同时下载多个网页
- 路径替换 — 完成后自动替换为本地相对路径，纯离线浏览
- 系统托盘 — 关闭窗口最小化到托盘
